Que es Angular?

Angular es un framework para crear apps web SPA (Single Page Aplication).

Características

Separa el Frontend y el Backend de la app
Simplifica el código
Tiene patron MVC
Basado en componentes
Codigo abierto
Usa el lenguaje Typescript (Mejora de JS, POO muy fuerte)

¿Por que usar Angular?

El desarrollo es rápido
Es modular(Codigo reutilizable)
Fácil mantenimiento
